# Sortman #

A basic sorting library to visualize common sorting algorithms seen in University Classes.

### Documentation ###

Algorithms take in an []int array and return the sorted array.

	u := []int{3, 2, 1, 6, 8, 5, 7}
	bubbleSort(u)
	insertionSort(u)
	selectionSort(u)
